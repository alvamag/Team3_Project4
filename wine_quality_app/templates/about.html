About This Project

DataVine 12 is a project driven by a passion for both data science and wine! We focused on predicting wine quality using machine learning, but made sure to keep the process enjoyable and creative. After all, who doesn’t enjoy a glass of wine while learning new skills? Our goal is to help both wine enthusiasts and professionals understand what makes a high-quality wine by exploring the relationship between characteristics like acidity, alcohol content, and pH. DataVine 12 uses machine learning to predict wine quality based on various physiochemical features. This project integrates data analysis, predictive modeling, and visualization, to provide insights into what makes a high-quality wine, leveraging the power of machine learning algorithms.

This project followed a full machine learning pipeline, from data retrieval and preprocessing to model development and evaluation. Using Spark SQL, we gathered and prepared the wine quality dataset, applying standardization and normalization techniques. We trained a supervised learning model using Scikit-learn, achieving over 75% classification accuracy. The modeling process included iterative optimization and performance tracking, all documented within the codebase. The final solution was designed to go beyond analysis — offering a more dynamic way to apply the model’s insights.

  • Python: For data analysis, modeling, and scripting
  • Pandas: For data cleaning and manipulation
  • Scikit-learn: For building and evaluating the machine learning model
  • Spark SQL: For querying and aggregating the dataset
  • Flask: For serving the model through a prediction API
  • HTML, CSS, Bootstrap: For front-end layout and styling
  • JavaScript: For form handling and dynamic UI behavior
  • Plotly & Leaflet: For interactive data visualizations
  • Tableau:For data exploration and visual storytelling
  • Gunicorn: For running the Flask app in a production environment
  • Render: For hosting and deploying the live web application
